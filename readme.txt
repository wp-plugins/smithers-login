Add style to your login page!! This plugin enables you to specify a style sheet to be used on the login page.

== Description ==

WordPress's themes are absolutly amazing, except you are not able to theme the login pages very well.
Smithers Login is a light weight plugin that will allow you full theme control over the login pages.

Some Features:

* Specify CSS style sheet for login pages.
* Change the URL and TITLE of the header link.
* Add specific messages to the different login pages (Login, Registration, Forgot Password)
* Change various error messages on login and registration pages.
* Enhances and extends DBD Login Style.

== Installation ==

1. Upload the `smithers-login` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Locate the 'Smithers Login' item on the 'Network Admin' menu
4. Change your options, and you are good to go!

== Frequently Asked Questions ==

= Where should I put my CSS file? =

Although you can host it where ever you would like, I reccomend placing it in your theme directory as `login-style.css`
